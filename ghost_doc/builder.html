Function scm->json scm

  • Prototype: (scm->json scm #:optional (port (current-output-port)) #:key (escape #f) (pretty #f)))
  • Description: Creates a JSON document from native. The argument @var{scm} contains the native value o f the JSON document. Takes one optional argument, @var{port}, which defaults to the current output port where the JSON document will be written.
  • Arguments:
    • port : (Optional) A port from which the output is generated
    • key : (Optional)
  • Return: A JSON document
