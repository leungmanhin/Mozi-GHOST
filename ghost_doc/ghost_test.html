Function find-about-gene

  • Prototype: (find-about-gene X)
  • Description: Finds a pattern, (MemberLink X (VariableNode "$a")) for the given gene and returns infromation about the gene's GO membership.
  • Arguments:
    • X : Gene for which information is needed
  • Return: The GeneCatagory (GO) the input geneNode belongs to

  • Arguments:

  • Return: All Genes related to the given GO catagory.

Function find-common-genes

  • Prototype: (find-common-genes x y)
  • Description: Retrives common genes that exist in different GOs.
  • Arguments:
    • x : GO term of the first gene catagory
    • y : GO term of the second gene catagory
  • Return: All genes that are present in both of the given GOs binded with a MemberLink.

Function find-cellular-comp

  • Prototype: (find-cellular-comp x)
  • Description: The input gene belongs to a GO that contains one or many other genes that also belong to different other GOs. This function finds all these GOs belonging to the GO namespace "Cellular component."
